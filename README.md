# 🌸 Мой Персональный Планировщик

Персональное PWA-приложение для iPhone.

## Как установить
1. Открой в Safari: https://polovinka587.github.io/personal-planner/
2. Нажми "Поделиться" → "На экран 'Домой'"
3. Готово! Приложение установлено

## Функции
- 📝 Задачи с категориями
- 🔔 Напоминания
- 💭 Вопросы для саморефлексии
- 🌙 Автоматическая тема
- 💾 Все данные на устройстве
